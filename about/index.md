# 关于本站

## 建站目的

这个网站用于记录和整理我在不同领域的学习笔记，主要包括：

- 🤖 **AI** - 人工智能、机器学习、深度学习
- 💻 **编程** - 前端开发、后端开发、工具链
- 📈 **金融** - 投资理论、量化交易

## 技术栈

本站使用以下技术构建：

| 组件 | 技术 |
|------|------|
| 框架 | [VitePress](https://vitepress.dev/) |
| 语言 | TypeScript |
| 包管理 | pnpm |
| 部署 | Cloudflare Pages |

## 内容规范

### 笔记格式

每篇笔记遵循以下结构：

```markdown
---
title: 标题
description: 简短描述
---

# 标题

## 概述

## 核心概念

## 代码示例

## 总结
```

### 代码示例

- 所有代码示例需要可运行
- 复杂代码需要添加注释
- 使用中文注释
