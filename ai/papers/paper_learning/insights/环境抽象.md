# 环境抽象

> **定义**: 将环境（聊天室、迷宫、网络）视为特殊的 Agent，统一交互协议，降低系统复杂度。

## 核心要点

- **环境即 Agent**: 聊天室、游戏场景也是"智能体"
- **统一协议**: Agent ↔ Environment 使用相同的 `reply()` 接口
- **双向通信**: Agent 发消息 → 环境 广播 → 其他 Agent 收到
- **嵌套环境**: 环境可以嵌套（房间内的房间）

## 示例

```python
# 将聊天室视为 Agent
class ChatRoom(Agent):
    def __init__(self):
        self.participants = []  # 加入聊天室的 Agent
        self.history = []

    def reply(self, msg):
        # 收到消息后广播给所有参与者
        self.history.append(msg)
        for agent in self.participants:
            agent.receive(msg)
        return "message broadcasted"

# Agent 加入环境
room = ChatRoom()
agent1.join(room)
agent2.join(room)

agent1.reply("Hello")  # 发送给 room
# → room.reply() 广播给 agent2
```

## 为什么重要

传统设计：
```
Agent → Environment API → Environment → Agent API → Agent
（两套 API，复杂耦合）
```

环境抽象：
```
Agent → reply() → Environment(Agent) → reply() → Agent
（一套 API，简洁统一）
```

**适用场景**:
- 聊天室、社交网络模拟
- 游戏世界、物理环境
- 多层嵌套场景（大楼→房间→桌子）
