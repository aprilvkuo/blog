## 🚀 快速开始

### 1. 环境准备

```bash
cd /path/to/autogen/python
uv sync --all-extras
source .venv/bin/activate
```

### 2. 运行示例

```bash
# 创建学习样本目录
mkdir -p learning_samples

# 复制代码到对应文件后运行
python learning_samples/01_hello_agent.py
python learning_samples/02_pubsub_agent.py
python learning_samples/03_routed_agent.py
python learning_samples/04_multi_agent_collab.py
```

## 📝 学习建议

### 第一步：理解基础（1-2 小时）
1. 阅读 [01-HelloAgent](01-HelloAgent.md)
2. 运行代码并观察输出
3. 修改代码实验不同行为

### 第二步：深入理解（2-3 小时）
1. 学习 [02-发布订阅](02-发布订阅.md) 和 [03-消息路由](03-消息路由.md)
2. 理解 RPC vs 广播的区别
3. 尝试添加新的路由规则

### 第三步：综合应用（3-4 小时）
1. 学习 [04-多 Agent 协作](04-多 Agent 协作.md)
2. 设计自己的多 Agent 系统
3. 实现特定场景的应用
