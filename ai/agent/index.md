## 关键洞察

> 框架层的算法确实简单：**Agent = LLM + tool call loop**

但框架解决不了的问题才是本质：

- **Planning 瓶颈** - long-horizon task decomposition 在模型层未解
- **Multi-Agent 协调** - 信息损耗、幻觉累积、自我认知缺失
- **Eval 飞轮** - Anthropic 的护城河，开源缺的是这个
