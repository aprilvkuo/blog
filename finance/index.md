# 金融投资概述

投资是将资金配置到预期能产生回报的资产的过程。

## 核心概念

### 资产类别

| 资产类别 | 风险 | 预期回报 | 流动性 |
|----------|------|----------|--------|
| 股票 | 高 | 高 | 高 |
| 债券 | 中 | 中 | 高 |
| 房地产 | 中 | 中 | 低 |
| 现金 | 低 | 低 | 高 |
| 大宗商品 | 高 | 可变 | 中 |

### 投资原则

1. **风险与回报成正比**
2. **分散投资降低风险**
3. **长期投资优于短期投机**
4. **复利是世界第八大奇迹**

## 估值基础

```python
# 现值计算
def present_value(future_value, rate, periods):
    """计算现值"""
    return future_value / (1 + rate) ** periods

# 例子：1 年后 110 元，年利率 10%，现值是多少？
pv = present_value(110, 0.10, 1)
print(f"现值：{pv}")  # 输出：100.0
```

## 常见指标

- **PE 比率**：股价/每股收益
- **PB 比率**：股价/每股净资产
- **ROE**：净资产收益率
- **股息率**：年度股息/股价
