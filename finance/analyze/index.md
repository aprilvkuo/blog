# 股票分析工具

使用 AI 分析股票基本面、技术面和市场情绪，生成专业的投资研究报告。

## 使用说明

1. **输入股票代码或名称** - 例如：`多氟多 `、`002594.SZ`、`贵州茅台`
2. **点击"开始分析"** - 任务将提交到队列
3. **等待分析完成** - AI 将分析市场数据、 sentiment、基本面等
4. **查看完整报告** - 分析完成后即可查看详细报告

## 分析报告内容

每份报告包含：

- **市场分析师** - 技术面和市场环境分析
- **情绪分析师** - 新闻和社交媒体情绪
- **基本面分析师** - 财务数据和估值分析
- **多方观点** - 看涨理由
- **空方观点** - 看跌理由
- **交易计划** - 具体的入场/出场策略
- **风险评估** - 风险因素和管理建议
- **最终决策** - 综合买卖建议

## 注意事项

- 分析可能需要 5-10 分钟
- 结果会自动保存到 [股票分析](/finance/stock-analysis/) 目录
- 本工具仅供参考，不构成投资建议
